# Vercel Deployment Commands

## Quick Commands (if you have Vercel CLI access)

### 1. Install Vercel CLI
npm install -g vercel

### 2. Login to Vercel
vercel login

### 3. Link to existing project
vercel link --project-id prj_iNxxSQNsajfaG6lfvozO8k9LMUkg

### 4. Deploy to production
vercel --prod

### 5. Deploy with specific environment variables
vercel --prod -e DATABASE_URL=file:/tmp/custom.db -e NEXTAUTH_SECRET=your-secret-here

- Project ID: prj_iNxxSQNsajfaG6lfvozO8k9LMUkg

## Required Environment Variables
DATABASE_URL=file:/tmp/custom.db
NEXTAUTH_URL=https://your-app.vercel.app
NEXTAUTH_SECRET=generate-a-32-character-secret

## Manual Deployment (Recommended)
1. Go to https://vercel.com/jiten-kumars-projects/know-your-tax
2. Click "Import Git Repository" or "Redeploy"
3. Add environment variables in Settings
4. Click Deploy
